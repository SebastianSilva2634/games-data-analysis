# Análisis de Tendencias en la Industria de Videojuegos 🎮

## 📌 Descripción del Proyecto
Este proyecto se enfoca en el análisis de datos históricos de ventas de videojuegos (1980-2016) para identificar patrones que determinen el éxito de una plataforma o género. El objetivo final es proporcionar información valiosa para la planificación de campañas publicitarias y estrategias de mercado para el año 2017.

## 🧪 Mi Rol y Metodología
Como **Analista de Datos**, realicé las siguientes etapas:
1. **Preparación de Datos:** Limpieza de nombres, manejo de valores ausentes y conversión de tipos de datos.
2. **Análisis Exploratorio (EDA):** Estudio del ciclo de vida de las consolas y las ventas por región (NA, EU, JP).
3. **Análisis Estadístico:** Realización de pruebas de hipótesis (T-test) para comparar las calificaciones de usuarios entre géneros y plataformas.
4. **Perfilado de Usuario:** Determinación de las preferencias regionales por género y clasificación (ESRB).

## 🛠️ Tecnologías Utilizadas
* **Python 3.x**
* **Pandas:** Manipulación y limpieza de datos.
* **Matplotlib & Seaborn:** Visualización de datos y diagramas de caja (boxplots).
* **SciPy:** Pruebas de hipótesis estadísticas.

## 📈 Conclusiones Clave
* Se identificó que el ciclo de vida promedio de una consola exitosa es de aproximadamente 10 años.
* Existe una correlación moderada entre las reseñas de los críticos y las ventas, pero no tanto con las reseñas de los usuarios.
* El género **Action** es el líder global en ventas, mientras que en Japón el género **Role-Playing** domina el mercado.
